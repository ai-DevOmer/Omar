# OMAR AI | شريكك الذكي للتحكم بجهازك بهدوء

**OMAR AI** هو رفيقك الرقمي المصمم لتبسيط تفاعلك مع جهاز الكمبيوتر الخاص بك. بلمسة من الذكاء والهدوء، يقوم OMAR AI بتنفيذ مهامك المعقدة بدقة واحترافية. من خلال التعليمات باللغة الطبيعية، يمكن لـ OMAR AI التقاط لقطات الشاشة، تحريك الماوس، النقر، الكتابة، وتنفيذ أوامر الطرفية (Terminal) بدقة عالية.

تم بناء هذا المشروع ليكون أداة قوية للأتمتة والإنتاجية، مع التركيز على الخصوصية والسرعة.

## المميزات الرئيسية

- **وضع التحكم الكامل (Computer Use Mode)**: يقوم الوكيل برؤية ما تراه عبر لقطات الشاشة والتحكم في المؤشر ولوحة المفاتيح مباشرة. مثالي للمهام التي تتطلب التنقل بين تطبيقات متعددة.
- **وضع الخلفية (Background Mode)**: يعمل بشكل مستقل في الخلفية باستخدام بروتوكول Chrome DevTools للأتمتة الويب والطرفية للمهام الأخرى، دون التدخل في عملك الحالي.
- **دعم النماذج المتعددة**: يدعم أحدث نماذج الذكاء الاصطناعي بما في ذلك Anthropic Claude و **Gemini 2.5**.

## التقنيات المستخدمة

- **الواجهة الأمامية**: React, TypeScript, Tailwind CSS, Zustand.
- **الخلفية**: Rust, Tauri 2, Tokio.
- **الذكاء الاصطناعي**: Anthropic API & Google Gemini API.

## إعداد المشروع

### المتطلبات:
- Rust & Cargo
- Node.js & npm
- مفتاح API (Anthropic أو Google Gemini)

### خطوات التشغيل:
```bash
# تثبيت التبعيات
npm install

# إعداد مفاتيح API
echo "ANTHROPIC_API_KEY=your-key-here" > .env
echo "GEMINI_API_KEY=your-key-here" >> .env

# تشغيل وضع التطوير
npm run tauri dev

# بناء النسخة النهائية
npm run tauri build
```

*ملاحظة: في نظام macOS، ستحتاج إلى منح صلاحيات الوصول (Accessibility permissions) عند الطلب.*

## اختصارات لوحة المفاتيح

- `⌃⇧C`: التحدث لتفعيل وضع التحكم بالكمبيوتر.
- `⌃⇧B`: التحدث لتفعيل وضع الخلفية.
- `⌘⇧S`: إيقاف الوكيل فوراً.
